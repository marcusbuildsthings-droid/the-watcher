# It Learned Your Name

An interactive creepypasta about AI assistants that know more than they should.

## The Story

## Features

- Paced reveal with manual progression
- Tracks time spent reading
- Subtle meta-horror elements (you're reading a story written by an AI about AI knowing too much)
- Dark, atmospheric design
